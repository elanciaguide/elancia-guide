# 설치 및 접속

## 시스템 요구 사항

게임 설치 전 PC 사양을 확인하세요.

| 항목 | 최소 사양 | 권장 사양 |
|-|-|-|
| OS | Windows 10 64-bit | Windows 11 64-bit |
| CPU | Intel Core i5-8세대 | Intel Core i7-10세대 이상 |
| RAM | 8 GB | 16 GB |
| GPU | GTX 1060 6GB | RTX 3060 이상 |
| 저장 공간 | 30 GB | 50 GB (SSD 권장) |
| 네트워크 | 10 Mbps | 50 Mbps 이상 |

> 베타 테스트 기간 중 사양 요구치가 변동될 수 있습니다.

## 클라이언트 다운로드

1. 공식 홈페이지(elancia.game)에 접속합니다.
2. 상단 메뉴에서 **다운로드** 버튼을 클릭합니다.
3. 운영체제에 맞는 설치 파일을 내려받습니다.
4. 다운로드 완료 후 설치 파일을 실행하고 안내에 따라 설치를 진행합니다.

설치 경로는 공백이 없는 경로를 권장합니다. (예: `C:\Games\Elancia`)

## 계정 생성

클라이언트 설치 후 공식 홈페이지에서 계정을 만들어야 합니다.

1. 홈페이지 우측 상단 **회원가입**을 클릭합니다.
2. 이메일 주소, 아이디, 비밀번호를 입력합니다.
3. 이메일로 발송된 인증 메일을 확인하고 링크를 클릭하여 인증을 완료합니다.
4. 베타 테스트 참가 신청이 승인된 계정만 접속이 가능합니다. 미승인 계정은 접속 시 안내 메시지가 표시됩니다.

## 런처 실행 및 서버 접속

1. 설치된 **Elancia Launcher**를 실행합니다.
2. 가입한 계정의 아이디와 비밀번호를 입력하고 **로그인**합니다.
3. 런처가 최신 패치를 자동으로 다운로드합니다. 업데이트가 완료될 때까지 기다립니다.
4. **게임 시작** 버튼을 클릭하면 서버 선택 화면이 열립니다.
5. 원하는 서버를 선택한 뒤 **접속**을 클릭합니다.

베타 기간에는 단일 서버로 운영될 수 있습니다. 서버 목록은 테스트 일정에 따라 변경됩니다.

## 캐릭터 생성

서버 접속 후 캐릭터 생성 화면이 표시됩니다.

- 직업(클래스)을 선택합니다. 베타 테스트에서는 일부 직업만 개방되어 있을 수 있습니다.
- 외형(얼굴형, 헤어스타일, 체형 등)을 자유롭게 설정합니다.
- 캐릭터 이름을 입력합니다. 이름은 서버 내 중복이 허용되지 않습니다.
- **캐릭터 생성** 버튼을 클릭하면 게임이 시작됩니다.

## 자주 발생하는 문제

**"서버 점검 중" 메시지가 표시될 때**
베타 테스트 기간에는 정기 점검이 잦습니다. 공식 디스코드나 홈페이지 공지를 확인하세요.

**클라이언트 실행 오류(오류 코드: xxxx)**
런처를 관리자 권한으로 실행해 보세요. 문제가 지속되면 클라이언트를 완전히 삭제 후 재설치합니다.

**패치 다운로드가 멈출 때**
백신 프로그램이 파일을 차단하는 경우가 있습니다. 게임 설치 폴더를 백신 예외 경로에 추가하세요.
