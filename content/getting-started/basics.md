# 기본 조작과 UI

## 캐릭터 이동

일랜시아는 키보드와 마우스를 조합하여 캐릭터를 조작합니다.

- **WASD** 또는 방향키로 캐릭터를 이동합니다.
- 마우스 우클릭을 누른 채 드래그하면 시점(카메라)이 회전합니다.
- **Shift**를 누른 채 이동하면 달리기(스프린트) 상태가 됩니다. 스태미나를 소모하므로 주의하세요.
- **Space**를 누르면 점프합니다. 일부 지형에서는 이중 점프가 가능합니다.
- **Ctrl + 방향키** 또는 **Alt를 누른 채 클릭**하면 해당 지점으로 자동 이동합니다.

## 카메라 조작

| 동작 | 방법 |
|-|-|
| 시점 회전 | 마우스 우클릭 드래그 |
| 줌 인 / 아웃 | 마우스 휠 스크롤 |
| 1인칭 전환 | 마우스 휠 최대 줌 인 또는 `V` 키 |
| 카메라 초기화 | `R` 키 (캐릭터 뒤로 정렬) |

카메라와 캐릭터 이동 방향이 연동되어 있어, 카메라를 돌리면 `W` 키의 전진 방향도 함께 바뀝니다.

## 기본 단축키

### 캐릭터 / 상태

| 단축키 | 기능 |
|-|-|
| `P` | 캐릭터 정보 창 열기 |
| `I` | 인벤토리 열기 |
| `K` | 스킬 창 열기 |
| `Q` | 퀘스트 일지 열기 |
| `M` | 지도 열기 |
| `B` | 장비 창 열기 |

### 전투

| 단축키 | 기능 |
|-|-|
| 마우스 좌클릭 | 기본 공격 / NPC 상호작용 |
| `1` ~ `0` | 스킬 슬롯 1~10 사용 |
| `` ` `` (백틱) | 스킬 슬롯 확장 바 토글 |
| `Tab` | 가장 가까운 적 자동 타겟 |
| `F` | 타겟 해제 |
| `T` | 보조 타겟 고정 (파티원 등) |

### 편의 기능

| 단축키 | 기능 |
|-|-|
| `Enter` | 채팅 창 활성화 |
| `Esc` | 메뉴 닫기 / 취소 |
| `PrintScreen` | 스크린샷 저장 |
| `Alt + Enter` | 전체화면 / 창모드 전환 |
| `F10` | UI 전체 숨기기 (스크린샷용) |

> 단축키는 설정 창(`Esc` → 단축키 설정)에서 원하는 키로 변경할 수 있습니다.

## 화면 인터페이스(UI) 구성

게임 화면은 크게 다섯 영역으로 나뉩니다.

### 1. 상태 바 (화면 좌측 상단)
현재 캐릭터의 **HP(체력)**, **MP(마나)**, **스태미나** 게이지가 표시됩니다. HP가 0이 되면 사망하며 가까운 부활 지점으로 이동합니다.

### 2. 미니맵 (화면 우측 상단)
현재 위치와 주변 지형, NPC, 파티원의 위치가 표시됩니다. 미니맵 우측의 `+` / `-` 버튼으로 축척을 조절할 수 있습니다.

### 3. 스킬 바 (화면 하단 중앙)
사용 가능한 스킬을 슬롯에 등록하여 단축키로 사용합니다. 슬롯에 스킬을 배치하려면 스킬 창(`K`)에서 원하는 스킬을 슬롯으로 드래그하세요.

### 4. 퀘스트 트래커 (화면 우측)
현재 진행 중인 퀘스트의 목표가 실시간으로 표시됩니다. 항목을 클릭하면 목표 위치로 자동 이동합니다.

### 5. 채팅 창 (화면 좌측 하단)
전체 채팅, 파티 채팅, 시스템 메시지 등이 탭으로 구분되어 표시됩니다. 탭을 우클릭하면 채널별 표시 여부를 설정할 수 있습니다.
