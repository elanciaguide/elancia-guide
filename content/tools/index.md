# 인터랙티브 도구

일랜시아의 복잡한 스탯 시스템과 빌드 조합을 직접 시뮬레이션할 수 있는 도구 모음입니다. 현재 베타 서비스 중으로, 도구들은 순차적으로 추가될 예정입니다.

## 예정된 도구 목록

### 빌드 계산기 `준비 중`
클래스·스킬 조합을 선택하고 최종 스탯 합산치를 즉시 확인합니다. 장비 세트와 버프 조건을 반영한 실전 빌드 검토에 활용할 수 있습니다.

### 데미지 시뮬레이터 `준비 중`
공격력, 크리티컬, 속성 저항 등 전투 변수를 입력해 예상 딜량을 계산합니다. 보스 공략 시 딜 검증 용도로 설계됩니다.

### 스탯 분배 도구 `준비 중`
레벨업 시 포인트를 어디에 배분할지 시뮬레이션합니다. 목표 스탯에 도달하기 위한 최소 포인트 경로를 제안합니다.

### 강화·합성 확률 계산기 `준비 중`
장비 강화 단계별 성공 확률과 기대 비용을 산출합니다. 재료 수급 계획을 세울 때 참고 자료로 활용할 수 있습니다.

### 파티 시너지 분석기 `준비 중`
파티 구성원의 클래스 조합을 입력하면 시너지 버프 적용 여부와 역할 밸런스를 분석합니다.

> 도구 개발 우선순위는 커뮤니티 피드백을 반영해 조정됩니다. 필요한 도구가 있다면 의견을 남겨주세요.
